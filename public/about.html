What ScrinDeck Does

The app is built for repeatable digital signage: upload media, choose layouts, assign content to each part of the screen, and keep the display device playing even when connectivity is unreliable.

Use full-screen, split-screen, and collage layouts so more than one media type can appear at once.

How to Use ScrinDeck

1

Open the display player

On the TV, kiosk, or media device, open /display.html and put the browser in full screen.

2

Sign in to the control panel

Open /control.html on your phone, tablet, or computer and sign in with the admin password.

3

Upload and choose media

Add image or video files, then place them into the active playlist or assign them to layout zones.

4

Pick a layout

Select a template, choose which media appears in each region, and adjust the collage background.

5

Configure the ticker

Set the time block, scrolling text, label, colors, speed, and schedule for the bottom overlay.

6

Leave the player running

Once media has loaded, ScrinDeck keeps cached playlist items available for repeat playback.

Operating Notes

  • Use stable Wi-Fi or Ethernet when first loading a new playlist so the display device can cache the files.
  • Keep large videos short and compressed when using free hosting with limited storage.
  • For production, use a strong admin password and durable storage for uploaded media and saved settings.
  • If a screen is replaced or the browser cache is cleared, open the display player online again to rebuild its cache.
